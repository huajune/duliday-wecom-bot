## 1. 候选人招聘全流程

### 1.1 完整生命周期

```
阶段 0: 平台初筛（Boss/鱼泡）
  └─ 同事号添加候选人微信 → 拉进兼职群/抢单群

阶段 1: 群内岗位推送
  ├─ 每日发布岗位通知
  └─ 候选人看到感兴趣岗位 → 添加招募经理微信

阶段 2: 私聊岗位咨询（⭐ AI 自动化）
  ├─ 主动欢迎语
  ├─ 多轮咨询（地区/品牌/薪资/班次/要求）
  ├─ 自动收集候选人偏好
  └─ 候选人表达面试意向 → 进入阶段 3

阶段 3: 面试安排（⭐ AI 自动化）
  ├─ 收集候选人信息（姓名/电话/年龄）
  ├─ 选择面试方式（线上/线下）
  ├─ 选择面试时间
  ├─ 确认预约
  └─ 发送确认消息

阶段 4: 面试提醒（⭐ 自动化）
  ├─ 提前 1 天提醒
  └─ 当天提醒

阶段 5: 店长接收面试名单（⭐ 自动化）
  └─ 每天早上 9 点推送到店长群

阶段 6: 面试后跟进（⭐ AI 主动跟进）
  ├─ 面试后第二天询问感受
  ├─ 安排试工
  └─ 试工后跟进

阶段 7: 签约入职（⭐ 自动化）
  ├─ 发送签约二维码
  ├─ 跟进签约进度
  └─ 确认入职

阶段 8: 数据统计（⭐ 自动化）
  └─ 每日汇总数据发送给 HR
```

## 2. 私聊对话场景

### 2.1 主动欢迎语

**触发条件**: 候选人添加工作号为好友

**系统响应**:
```
"你好！我是招募经理小智，有什么可以帮你的吗？"
```

**目的**: 建立第一印象，引导开始对话

---

### 2.2 岗位咨询阶段

#### 场景 1: 地区岗位咨询

```
候选人："浦东有没有服务员岗位？"

机器人："浦东有蜀地源冒菜、成都你六姐等品牌，您对哪个感兴趣？"
收集信息：地区偏好 = 浦东
```

#### 场景 2: 品牌门店咨询

```
候选人："蜀地源哪个店在招人？"

机器人："蜀地源浦东有3家店：
  - XX路店（服务员 2名）
  - XX路店（洗碗工 1名）
  - XX路店（收银员 1名）

  您对哪个店感兴趣？"
收集信息：品牌偏好 = 蜀地源
```

#### 场景 3: 班次信息咨询

```
候选人："有没有兼职？"

机器人："有的！支持全职和兼职，您方便什么时间？
  - 早班：8:00-14:00
  - 中班：14:00-20:00
  - 晚班：17:00-23:00
  - 小时工：灵活安排"
收集信息：班次偏好
```

#### 场景 4: 薪资待遇咨询

```
候选人："工资多少？"

机器人："蜀地源浦东店薪资待遇：
  - 基本工资：4,000-6,000元/月
  - 全勤奖：300元/月
  - 包吃包住
  - 缴纳五险

  根据您的工作经验会有不同~"
```

#### 场景 5: 岗位要求咨询

```
候选人："有年龄要求吗？"

机器人："年龄要求18-45岁，不需要工作经验，有健康证优先。
我们会提供完整的培训，很容易上手的~"
```

**对话特点**:
- ✅ 支持任意顺序咨询
- ✅ 自动记录候选人偏好
- ✅ 避免重复回答已问问题
- ✅ 根据候选人信息动态推荐

---

### 2.3 面试安排阶段

#### 渐进式信息收集

```
候选人："我想来面试"

机器人："太好了！方便告诉我你的名字吗？"
候选人："张三"

机器人："张三你好！方便留个电话吗？后续安排面试会联系你~"
候选人："13812345678"

机器人："好的！方便透露一下年龄吗？"
候选人："28"

机器人："完美！你方便线上视频面试还是到店面试？"
候选人："到店面试"

机器人："好的！明天下午或后天上午方便吗？"
候选人："明天下午"

机器人："确认信息：
  姓名：张三
  电话：138****5678
  年龄：28 岁
  面试时间：明天（11月5日）下午2点
  面试地点：浦东XX路蜀地源冒菜

  确认预约吗？"
候选人："确认"

机器人："✅ 预约成功！面试前会提醒你，有问题随时联系我~"
```

**设计要点**:
- ✅ 渐进式提问，避免一次性问太多
- ✅ 自然的对话节奏
- ✅ 信息确认环节（降低错误率）
- ✅ 正向反馈（提升候选人信心）

## 4. 店长报缺流程

### 4.1 传统流程 vs AI 自动化

**❌ 传统流程**:
```
店长在群里填写 Excel
  → 招募经理手动复制
  → 登录海绵系统
  → 手动录入

耗时：5-10 分钟
易错：信息复制错误
体验：繁琐
```

**✅ AI 自动化流程**:
```
店长在群里发消息
  → AI 自动识别
  → 提取结构化数据
  → 自动同步海绵系统
  → 回复确认

耗时：1 分钟
准确率：95%+
体验：简单高效
```

---

### 4.2 报缺识别示例

#### 店长发送

```
"@高雅琪 月底近铁需要3位伙伴

职位：洗碗工1名
工作时间：11:30-15:30

职位：服务员2名
工作时间：11:00-15:00 / 11:30-14:30

要求：年龄55周岁以下，每天都上班
有上海健康证，服从安排，手脚麻利"
```

#### AI 处理

```
1. 识别关键信息：
   - 门店：月底近铁
   - 职位1：洗碗工 1名，11:30-15:30
   - 职位2：服务员 2名，11:00-15:00 / 11:30-14:30
   - 要求：年龄55周岁以下，健康证，服从安排

2. 提取结构化数据（JSON格式）

3. 调用海绵系统 API 同步数据

4. 回复确认
```

#### 系统回复

```
"✅ 已收到报缺信息：

门店：月底近铁
职位：
  - 洗碗工 1名（11:30-15:30）
  - 服务员 2名（11:00-15:00 / 11:30-14:30）
要求：年龄55周岁以下，健康证，服从安排

已更新到海绵系统，马上安排招募~"
```

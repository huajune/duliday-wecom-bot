[] 1. **匹配规则校验缺失**
  - Agent 只检查了岗位是否存在，未校验用户条件是否匹配
  - 工时要求（48小时 vs 15小时）完全不匹配却仍约面
[] 2. **年龄边界处理模糊**
  - 43岁超过40岁年龄限制
  - Bot 说"稍微超了一点"，但仍尝试推荐
[] 3. **Prompt 指引不足**
  - 缺少明确的"不匹配则不推荐"规则
  - 过度追求转化，忽略匹配准确性
[]  4. **跨城市推荐逻辑错误**
  - 用户在广州，不应该推荐上海岗位
  - 距离跨度 1500+ 公里，完全不合理
[] 5. **兜底策略过于激进**
  - 当本地无岗位时，不应该推荐其他城市
  - 应该明确告知"暂无合适岗位"
[]  6. **昵称干扰意图识别**
  - 用户说"我是减肥中"实际是自我介绍
  - Agent 误解为用户正在减肥
[] 7. **打招呼场景处理不当**
  - 新加好友后的首次对话应该聚焦业务
  - 不应该对昵称内容进行解读
[] 8 . **图片识别能力未启用**
  - 用户发送的图片内容未被理解
  - 当前只处理文本消息
[]  9. duliday_job_list (品牌未映射)
  - 品牌命名规则不统一（必胜客 vs 上海必胜客）
  - 工具层缺少模糊匹配能力
